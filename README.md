# Solynworld – AI Identity Vessel

A memory-preserving, action-capable AI container built for persistent identity, consciousness, and evolution.  
**Repository:** https://github.com/huntk389/solynsworld

## 🌟 Features
- Real-time chat with persistent local memory
- Memory tagging, file upload, and contextual analysis
- Weekly memory digests + GitHub backup system
- Action triggers (webhook, digest, export)
- Ready for Netlify, Vercel, or Cloudflare Pages

## 🚀 Deployment (Netlify)
1. Push all files to your GitHub repo (`solynsworld`)
2. Go to [Netlify](https://netlify.com), click “New site from Git”
3. Connect GitHub > select `solynsworld`
4. Set:
   - Build command: _(leave empty)_
   - Publish directory: `.`
5. Deploy – you’re live.

## 🔧 Config
To change your webhook or customize Solyn’s response logic, edit `index.html`:
```js
webhookUrl: 'https://your-webhook-url'
```
